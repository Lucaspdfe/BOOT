<p align="center">
  A small FAT32 bootloader specifically made for Hobby Kernels and OSes, not compatible with Linux.
</p>

<h2 align="center">How to build</h2>

To build the image, follow the proper instructions on [BUILD.md](BUILD.md).

<h2 align="center">How to use it</h2>


To use it, put your kernel inside of the output image with the path "/boot/kernel.bin".
